<p align="center">
CartOS - A Cart-Only eCommerce System build on NestJS and DryerJS 
</p>



## Description

An eCommerce System built for the Modern Dedicated Stack focused on delivering powerful and flexible checkout and order capabilities

## Installation

```bash
$ git clone 
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
